DBR難易度表(イージー) … オーソドックスな難易度表です AC現行作品で遊べない曲は最下部に表示されます
DBR難易度表(イージー)(おすすめ) … よりおすすめ度の高い譜面のみを厳選して表示しています
DBR難易度表(イージー)(全譜面) … 逆に難易度査定済であればすべて表示します 曲数を求める方に
DBR難易度表(イージー)(INFINITAS) … 査定済全譜面から、INFINITASでプレーできる譜面のみを表示しています 曲の解禁方法も併せて記載しています
DBR難易度表(イージー)(新規提案有) … 査定済全譜面の内容に加え、管理用スプレッドシートで新規提案されている譜面を表示しています 更に曲数を求める方、難易度査定に協力していただける方に
DBR難易度表(イージー)(INFINITAS)(新規提案有) … 同じく新規提案分を追加したINFINITAS版です 特にAC削除済曲やINF限定曲はこちらで見た方が早いです
DBR難易度表(イージー)(未査定有) … 査定済全譜面に加え、管理用スプレッドシート内未査定の全譜面を表示します 新規提案の検討などにお使いください
管理用スプレッドシート … DBR難易度表(イージー)共通の管理用スプレッドシートです 譜面の新規提案、難易度やおすすめ度の変更提案、その他表運営に関する事柄はすべてこちらに

DBR段位(仮) … 決められた4曲を連続でプレーし、3曲以上アシストイージーで合格となる疑似段位です(作:@DBR_MEN様 掲載許可いただきありがとうございます)
DBR段位(仮)(INFINITAS) … INFINITASでプレーできる譜面のみを表示します 性質上4曲揃わない状態の段位が存在するので参考程度に
