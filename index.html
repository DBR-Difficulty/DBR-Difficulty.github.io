  • DBR難易度表について … 表の詳しいルール、説明を記載しています。不明な点があれば順次追記しますので、ご連絡ください。
DBR難易度表(イージー) … オーソドックスな難易度表です AC現行作品で遊べない曲は最下部に表示されます
DBR難易度表(イージー)(おすすめ) … 強くおすすめされている譜面のみを厳選して表示しています
DBR難易度表(イージー)(全譜面) … 逆に難易度査定済であればすべて表示します 曲数を求める方に
DBR難易度表(イージー)(INFINITAS) … 査定済全譜面から、INFINITASでプレーできる譜面のみを表示しています 曲の解禁方法も併せて記載しています
DBR難易度表(イージー)(新規提案有) … 査定済全譜面の内容に加え、管理用スプレッドシートで新規提案されている譜面を表示しています 更に曲数を求める方、難易度査定に協力していただける方に
DBR難易度表(イージー)(INFINITAS)(新規提案有) … 同じく新規提案分を追加したINFINITAS版です AC削除済曲やINF限定曲はこちらで見た方が早いです
DBR難易度表(イージー)(未査定有) … 査定済全譜面に加え、☆7以上のDBRでプレー可能（=新規提案可能）な全譜面を表示します 新規提案の検討、拡張機能使用時などにお使いください
DBR難易度表(20071224) … DBR難易度表（2007年最終更新）の内容を表示します 他表より低難易度の譜面が多く含まれます 新規提案の検討にお使いください
管理用スプレッドシート … DBR難易度表(イージー)共通の管理用スプレッドシートです 譜面の新規提案、難易度やおすすめ度の変更提案、その他表運営に関する事柄はこちら

DBR段位(仮) … 決められた4曲を連続でプレーし、3曲以上アシストイージーで合格となる疑似段位です(作:@DBR_MEN様 掲載許可いただきありがとうございます)
DBR段位(仮)(INFINITAS) … INFINITASでプレーできる譜面のみを表示します 性質上4曲揃わない状態の段位が存在するので参考程度に

拡張機能 … 上記各種表で使用できるPC版Google Chrome向け拡張機能です ブラウザにランプ、BP、スコアの保存が可能になります
DBRプレー記録管理 … 上記拡張機能と記録を共有しており、ここからもブラウザに保存が可能です 表示項目や機能は最低限ですが、拡張機能非対応ブラウザや、ゲームセンターなどでの使用に適しています

コラム … 主にDBR関連のちょっとしたお話

各種難易度表において、TexTage様へのリンクを掲載させていただいております。
特にDBRの正確なノーツ数を把握する上で不可欠なスクラッチ枚数の情報など、TexTage様のデータがなければこの難易度表は成立しませんでした。
心より感謝申し上げます。
